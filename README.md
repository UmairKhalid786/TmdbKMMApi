# TmdbKMMApi
A project built to provide basic TMDB Api using KMM


### Simply include shared project build in your project and you will be able to access the following use cases for both movies and series on TMDB

- Credit
- Content Detail
- Genres List
- Multi-Search
- Now Playing
- On-Air Content
- Popular Content
- TopRated Content
- Trending Content
- Upcoming
